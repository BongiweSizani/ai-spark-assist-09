
# Atlas - AI Workplace Productivity Assistant

### 1. Problem Statement
Office Administrators spend 5+ hours weekly on repetitive tasks: drafting emails, summarizing meetings, and prioritizing tasks. This reduces time for core admin work and increases errors from manual work.

### 2. Solution Overview  
Atlas is an AI copilot built for Office Admin role. It automates 3 key workflows: 
1. Tone-aware email drafting 2. Meeting notes → key points + deadlines 3. Task prioritization using Eisenhower Matrix

### 3. Tools Used
- **Lovable AI**: No-code app builder to generate prototype
- **GitHub**: Version control + documentation. Repo set to Private for data protection.

### 4. Sample Prompts Used
*Prompt 1 - Structure*: "Build an office admin dashboard for Operations role. Include 4 modules: Smart Email Generator with tone selector, Meeting Notes Summarizer that extracts key points, decisions, deadlines, AI Task Planner with Eisenhower matrix, and AI Research Assistant. Use clean, professional UI for beginners."

*Prompt 2 - Accuracy*: "For Meeting Notes Summarizer, create feature that takes raw meeting transcript and outputs: 1. 3 Key Points 2. Decisions Made 3. Action Items with deadlines. Add disclaimer: 'AI-generated content must be reviewed by human'."

### 5. Challenges + Solutions
| Challenge | Solution | Ethical Consideration |
| --- | --- | --- |
| AI hallucinated fake deadlines | Added disclaimer + forced human review | Accuracy + Human oversight |
| Risk of exposing client data | Set GitHub repo to Private | POPIA + Confidentiality |
| Generic AI outputs | Used structured prompts with specific outputs | Quality + Effectiveness |

### 6. Ethical Considerations & Limitations
1. **Human-in-the-loop**: All AI outputs must be reviewed before sending to clients. AI can make mistakes = "AI hallucination".
2. **Data Privacy**: Used demo data only. Repo is Private per POPIA. Never paste real client info.
3. **Limitation**: AI cannot replace human judgment for sensitive emails or legal decisions.

### 7. Features + Impact
**Impact**: Reduces admin time by ~2hrs/day on emails/notes.
